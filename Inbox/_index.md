# Inbox

> Temporary holding area for notes awaiting triage. **Status: 0 pending items**

## Routing Rules

Anything dropped here must be triaged within 7 days. Each note → one of:

1. **File** under `Projects/`, `Areas/`, or `Resources/` with a descriptive name.
2. **Link** from an `_index.md` or a daily note so it isn't orphaned.
3. **Archive** (move to `Archive/<original-path>/`) if it's done or stale.
4. **Delete** if duplicate or worthless.

If a note can't be classified in one read, write one sentence at the top explaining what it is — then file. The Inbox is a *transit* folder, not storage.

## Contents

Empty. All pending items have been processed.
