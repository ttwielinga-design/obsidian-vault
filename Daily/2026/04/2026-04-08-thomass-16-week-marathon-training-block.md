# Thomas's 16-Week Marathon Training Block
### Sub-3:30 Goal | Evidence-Based | Utrecht/Amsterdam | April 2026

---

## Overview

This block is synthesized from Domains 1–7 research and represents current best-practice marathon training for a fit 25-year-old recreational runner (Thomas) targeting sub-3:30. It uses a **pyramidal→polarized periodization shift**, integrates strength and plyometrics, full nutrition/recovery protocols, and builds to a proper 3-week taper.

**Race Week = Week 19 (3-week taper embedded in weeks 17–19)**

---

## Block Structure

| Phase | Weeks | Volume Range | Focus |
|-------|-------|-------------|-------|
| Base | 1–6 | 55–70 km/week | Aerobic foundation, strength base, run form |
| Build | 7–12 | 70–90 km/week | Lactate threshold, marathon pace, VO2max |
| Race-specific | 13–16 | 75–85 km/week → taper start | Polarized: hard/easy, MP runs, gut training |
| Taper | 17–19 | 60% → 50% → 30% of peak | Freshness, glycogen load, race prep |

## Weekly Integrated Systems Summary

### Strength Schedule (by Phase)

| Phase | Frequency | Session A | Session B |
|-------|-----------|-----------|-----------|
| Base (wks 1–6) | 2×/week | Heavy squats, RDL, hip thrust | Single-leg press, Nordic curl, Copenhagen |
| Build (wks 7–12) | 2×/week | Above + box jumps | Above + single-leg hops |
| Race-specific (wks 13–16) | 1×/week | Maintenance: 2 sets per exercise | Lower volume, maintain load |
| Taper (wks 17–19) | 0 | None | None |

### Sauna Protocol (Weeks 1–13)

- 3–5×/week, 15–20 min at 80°C
- Post-easy runs only
- Stop 4 weeks out from race (deload heat stress)

### Gut Training Integration

- Wks 1–4: Gel/fuel every 45 min on long runs
- Wks 5–8: Fuel every 30 min on long runs
- Wks 9–14: Fuel every 20–25 min on long runs (race simulation)
- Wks 15–19: Use exactly race-day products every 20 min on all runs >60 min

### HRV Protocol

- Daily morning measurement (Oura Ring recommended)
- Green = train as planned
- Amber (5–10% below avg) = downgrade to easy run
- Red (>10% below avg for 2+ days) = rest day inserted

## Key Performance Benchmarks to Track

| Metric | Baseline (Week 1) | Target (Week 14) |
|--------|------------------|-----------------|
| Easy run pace @ 65% HRmax | Record | Should be 10–15 sec/km faster |
| LT2 threshold pace (30-min TT) | Record | Should improve 15–25 sec/km |
| 10K time trial | Record | Aim: sub-43:00 |
| Long run fueling: gels per hour | Establish baseline | 3–4 gels/hr comfortably |
| Resting HRV (7-day avg) | Record | Should trend up or stabilize during taper |
