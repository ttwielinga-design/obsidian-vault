## Usage

Use Omnisearch or Smart Connections to find anything by topic; use the file tree to browse by date.

### Daily-note template

Templater is wired up to `Resources/Templates/daily.md` — every new daily note starts with **Tasks / Schedule / Log / Wins / Context** sections. The Context section is the load-bearing one: wiki-link people, decisions, and projects there to grow the graph.
