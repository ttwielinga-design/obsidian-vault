## Popularity RATIO Explanation

|  | Here is a BRIEF explanation how the POPULARITY RATIO works as reflected in this MENU-ENGINEERING-SHEET |  |  |  |  |  |  |  |  |
|----|----|----|----|----|----|----|----|----|----|
|  | To answer the question whether a dish is popular or not, the POPULARITY RATIO is used to measure and define how popular a dish is in the eyes of MANAGEMENT . |  |  |  |  |  |  |  |  |
|  | A POPULARITY RATIO is a percentage of the overall average POPULARITY INDEX and is set by MANAGEMENT. The POPULARITY NORM is a percentage of the average POPULARITY INDEX and classifies a dish as a Winner/Runner or a Sleeper/Loser. Any dish above the norm is either a Winner or Runner. Any dish below the norm is a Sleeper or a Loser. |  |  |  |  |  |  |  |  |
|  | If MANAGEMENT feels that a certain dish must be classified as popular, they will increase the POPULARITY RATIO. This is how it works: |  |  |  |  |  |  |  |  |
|  | Let us assume MANAGEMENT has set a POPULARITY RATIO of 25% . To classify a dish as popular the following formula is used: |  |  |  |  |  |  |  |  |
|  |  |  |  |  |  |  |  |  |  |
|  | (100% - popularity ratio) x average POPULARITY INDEX = POPULARITY NORM |  |  |  |  |  |  |  |  |
|  | In the example below the AVG POPULARITY INDEX = ( Total menu mix = 100% / number of dishes = 12) = 8,33% |  |  |  |  |  |  |  |  |
|  | ( 100% - 25% ) x 8,33% = 6,25. |  |  |  |  |  |  |  |  |
|  | Meaning that a dish is classified as popular if the POPULARITY NORM is higher than 6,25 . |  |  |  |  |  |  |  |  |
|  | Meaning: the HIGHER the POPULARITY RATIO is set, the SOONER a dish is classified as popular |  |  |  |  |  |  |  |  |
|  | as shown in the EXAMPLE below: |  |  |  |  |  |  |  |  |
|  | The average Gross Profit Margin in the example is: |  |  |  |  | 0.547 |  |  |  |
