**Liquidity ratios:\**
``` math
Current\ ratio = \ \frac{Current\ assets}{Current\ liabilities}
```

``` math
Acid\_ test\ ratio = \frac{Cash + marketable\ securities + notes + accounts\ receivable}{Current\ liabilities}
```

> 
> ``` math
> Working\ capital\  = \ Current\ Assets\ –\ Current\ Liabilities
> ```
>
> ``` math
> Working\ capital\ turnover\ ratio\  = \ \frac{Revenue}{Average\ Working\ Capital}
> ```
>
> ``` math
> {Accounts\ Receivable\ turnover\ ratio\  = \ \ \frac{Revenue}{Average\ Accounts\ Receivable}}
> ```
>
> ``` math
> Average\ collection\ period\  = \ \frac{Period\ }{Accounts\ Receivable\ turnover\ ratio}
> ```

**Solvency:**\
``` math
Solvency\ ratio\  = \ \frac{Total\ Assets}{\ Total\ Liabilities}
```

``` math
{Debt\_ equity\ ratio\  = \ \frac{Total\ Liabilities}{Owner’s\ Equity}}
```

``` math
Number\ of\ times\ interest\ earned\ ratio\  = \ \frac{Earnings\ before\ interest\ and\ tax}{Interest\ expense}
```

**\**

**Activity:\**
``` math
Inventory\ food\ and\ beverage\ turnover\  = \ \frac{Cost\ of\ F\& B\ used}{Average\ F\& B\ Inventory}
```

``` math
Property\ and\ equipment\ turnover\ (or\ Fixed\ asset\ turnover) = \ \frac{Total\ Revenue}{Average\ Property\ and\ Equipment}
```

``` math
{Asset\ turnover\  = \frac{\ Total\ Revenue\ }{Average\ Total\ Assets}}
```

**Profitability:**

``` math
Return\ on\ Assets\  = \ \frac{Net\ income}{Average\ Total\ assets}
```

``` math
Return\ on\ Equity\  = \ \frac{Net\ Income}{Average\ Owner’s\ Equity}
```

``` math
{Operating\ efficiency\ ratio\  = \ \frac{Gross\ operating\ profit}{Total\ Revenue}}
```
