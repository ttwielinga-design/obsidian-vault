Hi Thomas,

Leif here from TurboScribe.


If you want the best possible transcript quality, there are a few settings that can make a big difference. Here are the tips I recommend most often:


1) Choose the correct audio language
When uploading a file, make sure Audio Language matches the spoken language in your file. TurboScribe supports over 98 languages.



  



  


If the wrong language is selected, transcript quality can drop right away. This is one of the easiest ways to improve accuracy.


2) Start with Whale mode
TurboScribe has 3 transcription modes:



Whale: highest accuracy (default)
Dolphin: balance of speed and accuracy
Cheetah: fastest

For most files, I recommend starting with Whale.



  


3) Use Restore Audio for difficult recordings
If your file has background noise, distant speakers, or poor microphone quality, open More Settings and enable Restore Audio.



  


This uses AI to enhance speech and can sometimes improve transcription accuracy when you&#x27;re having trouble with a file.


4) Turn on Speaker Recognition for conversations
If you&#x27;re transcribing an interview, meeting, podcast, or group discussion, enable Speaker Recognition.


Try setting the speaker count to Detect Automatically. This can improve speaker differentiation, though it can sometimes overestimate the number of speakers.



  


5) If needed, try another transcription mode
Whale is usually the best choice for accuracy, but different modes can produce different transcripts.


If a file still isn&#x27;t coming out quite right, it can be worth retranscribing it with Dolphin or Cheetah to compare the results.
