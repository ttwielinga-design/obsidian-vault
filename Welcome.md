## Filing Rules

- **Operational events** (meetings, calls, decisions made today) → today's daily note `## Log` section.
- **Stable reference facts** → appropriate `Resources/` or `Areas/` note. Link from daily log.
- **System fixes / lessons learned** → `Resources/Reference/issues-fixes-log.md` (append-only).
- **Unknown / not-yet-classified** → `Inbox/` with a clear filename.
- **Always wiki-link** people, projects, and prior decisions in daily notes.
- **Never delete** — archive to `Archive/<original-path>/` instead.
